This website is an online report calculator of the RP 1560. RP is a facebook powered roleplay. an active community of gamers. The concept of RP is to create an historical of fictional background where each player can take a nation/tribe/something and lead it to conquer other players, make alliaces and trade. The game base is represented by reports. Each report summarize a set of features of each nation (Soldiers, trade, money etc...). The operation of making reports is quite time consuming. It's easy to cheat and difficult to check if someone is cheating. In order to solve these problem I started experiments on an automatic calculator. My project is to create a form where to manually put all values. Then a JS will run a/some function and create the report.
